Graphic Spitz Emulator

QEMU patch with python GUI (LED/Button/UART)

Download this project as a .zip file Download this project as a tar.gz file

Patched qemu-system-arm that enables basic external parts (leds, switches, external UART) on "spitz" machine, plus a simple python GUI interface.

Copyright Dung Le 2013

Currently the GUI has:

  • button on GPIO 73, LED on GPIO 67
  • external UART (base addr: 0x1000_0000) with output interrupt on GPIO 10

The script establishes two telnet connections with qemu: one with its monitor and one with its virtual serial output. Led and switch actions are monitored using the custom hmp command "info ssbinfo".In addition, switch action is simulated by the "sendkey" command.

Interface:

  • Button: send button press event
  • Halt: stop the machine
  • Resume: resume the machine
  • restart: kill and restart emulator
  • reset: currently NOT supported since qemu does not have clean reset yet
  • LED output display
  • serial output display
